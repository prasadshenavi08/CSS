CSS Selecctors

CSS selectors specified some style for specified element. there are 5 types of selector.

  1. universal selector => it uses * to specify style
  2. class selector => it uses . to specify style
  3. id selector => it uses to specify style
  4. element selector => it uses element name to specify style
  5. sudo selector => it uses like hover, visited etc.
